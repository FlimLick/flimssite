Carrier

This is Carrier. A chat program that has been made from the ground up from nothing. It is a UDP dependant chatting aplication that includes sending images, making group chat, server based hosting, and soon to be voice calls. This was a project I made because I was simply board in the middle of math class.

The status of Carrier is alright. I meen after all I did make this in math class then continue it becuase I put to much effort into it. It does have G-Zip compression and server based encryption but there are some bugs a security issues that need to be assesed.

Drone Stuff

Let me introduce you to Drone Stuff.

It's a little game that I also made when I was bored and have a lot of freetime. Its based off of a Garry's mod map called gm_bigcity. I was really trying to challenge myself to import a map from a completely incompatible game engine to into the game I personally use (Godot 4.0)

I added the drone into the game because I was trying to figure out how impulse physics work in the game engine for Photon Spaceway. I like making other projects just to learn how a little function works only just to over complicate it and make a whole game based off that one function.

It does have multiplayer and modding capabilities but it is really buggy. I also made a song for the relatively nice menu I built for it.
